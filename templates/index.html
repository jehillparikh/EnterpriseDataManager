Enterprise Database API

A powerful and flexible API for managing users and questions in your enterprise application.

Built with Python Flask, this API provides a clean and RESTful interface for CRUD operations.

Quick Start

# Example API request with curl
curl -X POST http://localhost:5000/api/users \
  -H "Content-Type: application/json" \
  -d '{"username": "johndoe", "email": "john@example.com", "password": "securepass"}'

# Get all users
curl http://localhost:5000/api/users

# Create a question
curl -X POST http://localhost:5000/api/questions \
  -H "Content-Type: application/json" \
  -d '{"title": "How to use the API?", "content": "I need help using this API...", "user_id": "user-id-here", "tags": ["api", "help"]}'
